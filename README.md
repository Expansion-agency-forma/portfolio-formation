# Expansion Agency — Portfolio

## Stack
React 18 + Vite + Supabase + ESLint.

## Démarrage

```bash
npm install
cp .env.example .env   # puis remplir avec les creds Supabase de l'agence
npm run dev
```

## Scripts
- `npm run dev` — serveur de dev Vite
- `npm run build` — build production
- `npm run preview` — preview du build
- `npm run lint` — ESLint

## Structure
- `src/` — app React (components, hooks, lib, styles)
- `public/` — assets statiques servis tels quels
- `LANDINGPORTFOLIO/` — site statique HTML/CSS séparé
- `projet/` — ressources par projet client
- `CLAUDE FILES/` — contexte projet (MEMORY, PORTFOLIO, VISUAL SYSTEM)
